ext-imageannotation example

query parameters

bkgd_url points to the image to be annotated

imageannotation_text points to a json file that contains the text

url points to the url of an svg file that describes regions in the image. it should have the same dimensions as the image file. this is the only file that will be written back to.

svg-editor.html
?bkgd_url=ext-imageannotation-example/Hs-18480-facsimile.jpg
&url=ext-imageannotation-example/Hs-18480-annotations.svg
&imageannotation_text=ext-imageannotation-example/Hs-18480-lines.json
